[ CineAI 독립 실행 및 채점 가이드 ]

이 가이드는 AI(안티그래비티 등)의 도움 없이 로컬 웹 서버 환경에서 사이트를 완전히 독자적으로 구동하기 위한 안내입니다.

1. 준비 조건
   - PC에 PHP 8.x 이상 버전이 설치되어 있어야 합니다.
   - (만약 데이터베이스 초기화를 원하실 경우에만) PHP CLI 환경 및 기본 SQLite 모듈이 필요합니다.

2. 데이터베이스 초기화 (최초 1회만 실행)
   - 프로젝트 루트 폴더 내의 'setup_database_once.bat' 파일을 더블 클릭하여 실행합니다.
   - 콘솔 창에 "Database setup complete!" 문구가 출력되면 데이터베이스 초기화 및 테이블 생성이 성공적으로 완료된 것입니다. 창을 닫아주십시오.
   - 기본적으로 테스트용 데이터베이스 파일('movie_reviews.sqlite')이 프로젝트 폴더에 동봉되어 있어 바로 실행하셔도 무방합니다.

3. 서버 실행 (매 접속 시)
   - 프로젝트 루트 폴더 내의 'start_cineai.bat' 파일을 더블 클릭하여 실행합니다.
   - 콘솔 창에 "CineAI is running at http://localhost:8000" 문구가 나타나면 정상 구동 중인 것입니다.
   - 주의: 사이트를 테스트하는 동안 이 배치 파일의 검은색 콘솔 창을 종료하시면 안 됩니다. (최소화해 두시기 바랍니다.)

4. 브라우저 접속
   - Chrome 또는 Edge 등 웹 브라우저를 실행합니다.
   - 주소창에 아래 주소를 입력하고 접속합니다:
     http://localhost:8000

5. 문제 해결 (서버가 즉시 종료될 경우)
   - 'check_error.bat' 파일을 실행하여 오류 원인을 파악할 수 있습니다.
   - 주로 Windows 환경에 "Visual C++ 재배포 가능 패키지(VC++)"가 설치되지 않았거나 PHP 환경 변수가 등록되지 않은 경우일 수 있습니다.
