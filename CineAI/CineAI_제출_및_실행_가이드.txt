======================================================================
📢 [교수님 채점용] CineAI 웹 어플리케이션 실행 및 구동 가이드라인
======================================================================

본 프로젝트는 "보안 관제 센터(Security Monitoring Center)" 컨셉의 영화 리뷰 플랫폼으로,
교수님 컴퓨터에 **XAMPP(APACHE + MySQL + PHP) 환경이 설치되어 있어야 구동이 가능**합니다.

다음 두 가지 구동 방법 중 편하신 방법을 선택하여 실행해 주시기 바랍니다.

----------------------------------------------------------------------
[필수 사전 요구사항]
----------------------------------------------------------------------
* 교수님 PC에 **XAMPP**가 설치되어 있어야 합니다.
* 본 프로젝트의 자동 구동 배치 파일(.bat)은 XAMPP가 기본 경로인 [ C:\xampp ]에 설치되어 있음을 가정하고 작동합니다.

----------------------------------------------------------------------
방법 ① : XAMPP 경로 기반 원클릭 자동 구동 (가장 빠르고 간편함)
----------------------------------------------------------------------
XAMPP가 C:\xampp 경로에 설치되어 있다면, 폴더 내의 배치 파일을 통해 MySQL 서버 실행, DB 임포트, PHP 웹 서버 구동을 클릭 한 번으로 일괄 자동 처리할 수 있습니다.

단계 1) 데이터베이스 자동 생성 및 테이블 임포트 (최초 1회만 실행)
  - 프로젝트 폴더 내의 [ setup_database_once.bat ] 파일을 더블 클릭하여 실행합니다.
  - 자동으로 MySQL을 임시 구동하고, `movie_reviews_db` 데이터베이스를 생성한 뒤, 동봉된 `movie_reviews_db_perfect.sql` 데이터를 임포트합니다.
  - 콘솔 창에 "Database setup complete!" 문구가 보이면 성공입니다. 창을 닫아주세요.

단계 2) CineAI 서버 실행
  - 프로젝트 폴더 내의 [ start_cineai.bat ] 파일을 더블 클릭하여 실행합니다.
  - 자동으로 XAMPP 내의 MySQL 서버와 PHP 내장 서버를 8000포트로 한꺼번에 구동합니다.
  - 콘솔 창에 "CineAI is running at http://localhost:8000"이 표시되면 성공입니다.
  - ★ 주의: 채점하시는 동안 이 검은색 콘솔 창을 종료하지 마시고 최소화해 두시기 바랍니다.

단계 3) 웹 사이트 접속
  - 브라우저를 열고 주소창에 아래 주소를 입력하여 접속합니다:
    👉 http://localhost:8000

----------------------------------------------------------------------
방법 ② : XAMPP Control Panel 및 htdocs 폴더를 이용한 표준 구동
----------------------------------------------------------------------
기존의 전통적인 Apache 웹 서버 및 phpMyAdmin 관리자 페이지를 활용해 수동으로 구동하는 표준 방법입니다.

단계 1) 프로젝트 소스코드 폴더 이동
  - 제출받으신 [ CineAI ] 폴더 전체를 XAMPP의 htdocs 경로로 복사합니다:
    경로 예시: C:\xampp\htdocs\CineAI

단계 2) XAMPP Control Panel을 통한 서비스 시작
  - XAMPP Control Panel을 실행합니다.
  - **Apache**와 **MySQL** 모듈의 [Start] 버튼을 각각 눌러 활성화시킵니다.

단계 3) 데이터베이스(DB) 생성 및 데이터 구축
  - 웹 브라우저를 열어 phpMyAdmin 페이지에 접속합니다:
    👉 http://localhost/phpmyadmin
  - 좌측 상단 [새로 만들기(New)]를 클릭하여 데이터베이스명을 [ movie_reviews_db ]로 입력하고 생성(Create) 버튼을 누릅니다.
  - 생성된 데이터베이스를 클릭한 후, 상단 메뉴의 [가져오기(Import)] 탭으로 이동합니다.
  - [파일 선택(Choose File)] 버튼을 눌러 프로젝트 폴더 안에 있는 [ movie_reviews_db_perfect.sql ] 파일을 지정합니다.
  - 화면 맨 아래의 [가져오기(Import) / 실행] 버튼을 눌러 데이터 구축을 완료합니다.

단계 4) 웹 사이트 접속
  - 웹 브라우저 주소창에 아래 주소를 입력하고 접속합니다:
    👉 http://localhost/CineAI
